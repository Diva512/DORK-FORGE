# DORK FORGE v5 — Compact Cyber Dashboard Edition

Frontend-only static OSINT dork/query generator. Open `index.html` directly in a modern browser.

## What changed in this compact edition

- Enter Target area made smaller and cleaner.
- Analysis panel made compact so it does not take too much vertical space.
- Dork Variants panel gets more room on desktop.
- Light mode redesigned as a clean white cyber-lab theme.
- Dark cyber-hacker theme preserved.
- Runtime-only history preserved.
- No server, no npm, no database, no localStorage.

## Files

```txt
index.html
style.css
script.js
data.js
README.md
```

## Usage

1. Open `index.html` in a browser.
2. Enter a target, topic, company, or domain.
3. Choose engine/mode/category if needed.
4. Click Generate or Generate & Search.
